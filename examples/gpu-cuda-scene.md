# Example Scene: CUDA Opened a Door for GPU Programming

## Narration

CUDA opened a programming door for the GPU.

## Visual Description

On the left, show a simple GPU chip icon.

A cable comes out from the chip.

Next to the cable, show many 0s and 1s to represent binary computation and programming.

Keep the scene simple. The viewer should understand the metaphor in three seconds:

```text
GPU chip → programming path → binary computation
```
