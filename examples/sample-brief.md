# Sample Brief

Create a 45-second beginner-friendly video explaining what a terminal is.

Audience:
People who use AI tools and want to start coding, but feel intimidated by developer tools.

Tone:
Clear, warm, practical, not condescending.

Goal:
Help the viewer understand that the terminal is a text-based way to communicate with the computer, and that tools like Git, package managers, and Codex often use it as part of the workflow.

Visual style:
Clean educational interface, simple diagrams, warm colors, no dark hacker aesthetic.
